== Description ==

This plugin add a "Publier sur Facebook" button in the post modifing page.

== Installation ==


1. Upload the folder `publish-on-facebook` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Check if the box 'Publish on Facebook' apeared on the post-editing interface.

== Frequently Asked Questions ==

= Why does the button doesn't show on the meta box? =
It might be because the post has to be publish first on your blog before being send to your facebook wall.
